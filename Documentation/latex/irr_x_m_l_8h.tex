\subsection{Detailed Description}
Header file of the irrXML, the Irrlicht XML parser. 

This file includes everything needed for using irrXML, the XML parser of the Irrlicht Engine. To use irrXML, you only need to include this file in your project:



\begin{Code}\begin{verbatim}        #include <irrXML.h>
\end{verbatim}
\end{Code}



It is also common to use the two namespaces in which irrXML is included, directly after including \hyperlink{irr_x_m_l_8h}{irrXML.h}:



\begin{Code}\begin{verbatim}        #include <irrXML.h>
        using namespace irr;
        using namespace io;
\end{verbatim}
\end{Code}
